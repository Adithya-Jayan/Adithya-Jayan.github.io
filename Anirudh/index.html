My skills

My creative skills & experiences.

My skills include game development, web development, art development (both 3D and 2D), making music and coding.
I started making games during the 2020 lockdown. I got into this subject when my brother made a game using Unity following a brackeys “how to make a game” tutorial. The first game I made was by following imphenzia’s “The most basic tutorial I’ll ever make” video on YouTube. I made the models using Blender, which I started learning a few months before Unity. Again I was inspired by my brother. I found Sketchbook while searching for a good drawing app for an IPad. My brother taught me the keyboard and I got the guitar as my 13th birthday gift. I learnt QBasic as a topic in computer science and learned HTML and C++ from a course in code academy. Audacity, LMMS and Python I learnt from my brother.
